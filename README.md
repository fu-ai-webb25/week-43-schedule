###### Utveckling av AI-drivna applikationer, vecka 3 av 4

## Introduktion

Språkmodeller är kraftfulla, men deras kunskap är begränsad till det material de tränats på. Den här veckan lär vi oss hur AI kan kompletteras med externa datakällor genom Retrieval-Augmented Generation (RAG).

Vi bygger lösningar där språkmodellen hämtar information från dokument och databaser innan den genererar sitt svar. På så sätt kan vi skapa AI-applikationer som arbetar med aktuell och domänspecifik information.

## Mål för veckan:

1. Förstå hur Retrieval-Augmented Generation (RAG) fungerar.
2. Förstå hur vektordatabaser används tillsammans med språkmodeller.
3. Skapa embeddings från dokument.
4. Lagra dokument i en vektordatabas.
5. Integrera LangChain med en vektordatabas.
6. Bygga en enklare RAG-applikation.
